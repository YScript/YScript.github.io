1.1 Data Type Of JS Variables

JavaScript has six major type of data:

      number
      string
      boolean
      NULL
      undefined
      object

data type is defined as Primitive type and Object type
     type number is the type of data, type string is the type of string type data, type boolean is the type of bool data.
     type NULL is used to means An object that does not yet exist,Commonly used to represent a function attempting to return a non-existent object
     the Undefined type of data have one value only--undefined,when the declared variable has not been initialized, it would be set as a default value --undefined;
